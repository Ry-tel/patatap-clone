Hello user! This website is a simple clone of an awesome website patatap.com. It was build using paper.js for animations and howler.js for playing sounds. Press any button with a letter on a keyboard to create a circle and play corresponding sound. Numeric key will create a square with sound. The sounds overlaps, you can create your own "music" by hitting keys simultaneously.

**MY ADVISE** turn the volume down; especially if you are using headphones.

If you see the circles, but cannot hear the sound give the site 10 seconds to load the mp3s. Close this message and have fun!
